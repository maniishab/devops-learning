# DevOps Learning Practice


## Objective

To understand basic DevOps workflow and automation concepts.

## Topics Covered

### Git & GitHub
- Created a GitHub repository
- Practiced Git commands
- Cloned repository locally
- Checked repository status
- Used pull, add, commit and push commands

### Jenkins CI

- Installed and configured Jenkins
- Created a Jenkins job
- Connected Jenkins with GitHub repository
- Pulled code from GitHub using Jenkins
- Executed build steps successfully

## Tools Used

- Git
- GitHub
- Jenkins

## CI Workflow

Developer changes code  
↓  
Push changes to GitHub  
↓  
Jenkins fetches latest code  
↓  
Build process runs  
↓  
Build success

## Current Learning

Learning DevOps tools, automation and CI/CD concepts.
